The online version of the Jikes RVM userguide tracks the svn head of the project.

For releases of Jikes RVM, we export this userguide as a pdf file and as standalone html. Unfortunately, at the time of the 2.9.1 release, confluence exports on Codehaus were not working. Therefore, there is no exported user guide included in the 2.9.1 release. We direct you to either the 2.9.0 release, which did include an exported userguide, or the the online version.
